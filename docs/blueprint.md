# **App Name**: Guardian Command

## Core Features:

- Live Operational Map: An interactive map displaying active incidents, real-time officer locations (GPS tracked), AI-generated high-risk zones, patrol routes, and emergency escalation areas, providing a constantly updating visual overview.
- AI Video Intelligence: Real-time analysis of citizen-uploaded videos and officer bodycam streams using AI tools to detect weapons, violence, suspicious behavior, fire/smoke, and output threat levels, detected objects, and confidence scores.
- Automated Dispatch Engine: An AI tool that intelligently identifies and automatically assigns the nearest available officer to incidents, considering factors like traffic, officer workload, and incident severity to calculate ETA. Dispatchers can override assignments.
- Officer Safety Monitoring: Real-time tracking of officer status (Standby, Enroute, At Scene), heart rate, movement detection, and panic alerts, triggering automated alerts when an officer stops moving, signal is lost, or a panic button is activated.
- Predictive Crime Engine: An AI tool that uses historical and real-time data to identify crime hotspots, predict high-risk times, and recommend optimal patrol routes for proactive policing.
- Natural Language Command: An AI tool allowing commanders to interact with the system using simple voice or text commands, such as 'Show all high-risk incidents' or 'Track Officer 247'.

## Style Guidelines:

- Primary brand color: A deep, authoritative blue (#1414B8), evoking intelligence and precision, with high contrast on the dark theme.
- Background color: A very dark, subtle bluish-gray (#14141A), providing a clean, 'glassmorphic' base for a high-contrast dark theme.
- Accent color: white and background glassmorphic gray-400 1.50, providing a sharp highlight for interactive elements and critical alerts without clashing with the primary blue.
- Headlines and prominent text: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and assertive presence, suitable for quick information parsing.
- Body text and detailed information: 'Inter' (sans-serif) for its neutral, highly readable, and objective qualities, ensuring clarity across various data points.
- Sleek, minimalist, and high-contrast vector icons designed for rapid identification within a dark, information-dense interface. Icons will communicate status and actions with clarity.
- A 'glassmorphic' user interface design with modular, intelligently organized information panels against a dark background. Emphasis on clear hierarchy, intuitive navigation, and minimal clutter to facilitate high-stakes decision-making.
- Smooth and subtle animations, following the principles of Framer Motion, for dynamic data updates, system alerts, and transitions, enhancing the sense of a 'live operational brain'.
